# 사용자 매뉴얼 — investment-dashboard

> 기준 커밋: `9e8a85b` · 최신 main

처음 사용자를 위한 화면별 작동 가이드. 이 문서를 보면 모든 기능을 어떻게 쓰는지 알 수 있다.

---

## 1. 처음 시작 5분

### 1.1 카카오 연결 (1회만)
1. 하단 탭바의 **⚙ 설정** 또는 `/login` 접속
2. 노란 **카카오로 연결하기** 버튼 클릭
3. 권한 동의 — `talk_message` (나에게 보내기) 필수 체크
4. "연결 완료 ✓" alert 확인
5. 이후 모든 자동 알림이 본인 카카오톡으로 수신됨

### 1.2 현금 입력 (필수)
- `/dashboard` 상단 pill 줄에 **💰 현금 미입력** 노란 배지
- 클릭 → 입력 필드 → 예수금 원 단위 숫자 (예: `5000000`) → **저장**
- 입력 안 하면 비중 경고가 과대 발생 (보유 종목만으로 계산되어)

### 1.3 첫 종목 등록
1. `/dashboard` 우하단 **+ 플로팅 버튼** 클릭
2. Sheet 가 아래에서 슬라이드 인
3. 입력:
   - **종목 검색** — "삼성전자" 또는 "005930" → 자동완성 클릭
   - **평균단가** + **수량** (필수)
   - **손절선** + **익절선** (선택, 강력 권장)
4. **추가** 클릭 → 토스트 알림

### 1.4 첫 분석
- 보유 종목 카드 → **상세 분석** 또는 종목명 클릭
- 종목 상세 페이지 → **다시 분석** 버튼
- 1~2초 후 Claude 권고 + 손절·익절 표시

## 2. 화면별 가이드

### 2.1 상단 헤더 (공통)
모든 페이지 최상단에 sticky 헤더:
- 좌: **잃지 않는 투자** 로고 (홈으로) + 현재 페이지명
- 우: **🔔 알림 종 아이콘** — 클릭 시 우측 Sheet 슬라이드, 최근 7일 알림 표시
  - 빨간 점 = 미확인 알림 (localStorage 기준)
  - 필터 탭: **전체 / 보유 종목 / Pick**
  - 각 알림 클릭 시 해당 종목·스크리너로 이동

### 2.2 하단 탭바 (모바일)
4탭 항상 표시:
| 탭 | 경로 | 설명 |
|---|---|---|
| 🏠 포트폴리오 | `/dashboard` | 보유 종목 관리 |
| 📈 스크리너 | `/screener` | AI 추천 종목 |
| 🔍 종목찾기 | `/` | 빠른 분석 / 랜딩 |
| ⚙ 설정 | `/login` | 카카오 연결 |

### 2.3 대시보드 `/dashboard`

#### 상단 상태 pill 줄 (1초 체크)
한 줄에 핵심 요약:
| Pill | 의미 |
|---|---|
| 보유 N개 | 등록된 종목 수 |
| **총자산 X.X억원** | 보유 평가액 + 현금 |
| 💰 현금 X원 | 클릭 시 인라인 편집 |
| 오늘 ±X.XX% | 평가금액 가중 일간 수익률 |
| 피크 대비 -X% | 90일 피크 대비 하락률 (보일 때만) |
| 🟡 현금 비중 낮음 | 현금 < 10% 일 때 경고 |
| 🔴 즉시 확인 N건 | 손절·익절 도달 |
| 🟠 근접 N건 | 손절·익절 ±3% 이내 |

#### 포트 건강 배너 (있을 때만)
- **포트 낙폭 경고** — 피크 대비 -10% 이상
- **포트 낙폭 주의** — -7~-10%
- **비중 초과 N건** — 단일 종목 총 자산 25% 초과 (현금 포함 분모)

#### 손절/익절 도달 배너 (있을 때만)
긴급한 종목 리스트 + 현재가/손절/익절 가격

#### 포트폴리오 요약 (PortfolioSummary)
- 총 평가금액 + 원가 + 미실현 손익
- 총 수익률 + 오늘 가중 수익률
- KODEX 200 / KODEX 코스닥150 일간 변동 비교

#### 보유 종목 목록
각 카드 (접힌 상태):
- 좌: 종목명 + 티커 + AI 추천 배지 ("AI: 보유") + 알림 배지 (손절 근접/도달)
- 큰 글씨: **현재가** + 등락률 색
- **진행바**: 손절 ━━●━━ 익절 (현재가 위치)
- 우: **수익률 + 손익 금액** (등락 색)
- 좌측 경계선 색: 알림 레벨 (파란=손절도달, 빨강=익절도달 등)

카드 클릭 시 펼침:
- 평단 / 수량 / 평가 / 손절 / 익절 / AI 의견 / 분석일
- 버튼: **+ 추매 / ✎ 수정** (Sheet 열림) · **상세 분석** · **삭제**

#### 추매/수정 Sheet
**+ 추매** 탭 (기본):
- 추매 수량 + 추매 가격
- 미리보기: 새 수량 / 새 평단 / 평단 변화%
- **추매 확정** → 가중 평균 자동 계산

**✎ 직접 수정** 탭:
- 평단 / 수량 / 손절 / 익절 자유 편집
- 빈칸 저장 = 손절/익절 해제

#### 종목 추가 FAB
- 우하단 ⊕ 플로팅 버튼
- 클릭 시 Sheet 슬라이드
- 스크리너 "담기" 로 들어오면 자동 오픈 + 값 미리 채움
- 입력 중 가드레일 경고:
  - 집중도 25% 초과
  - 당일 +5% 이상 급등 (추격매수 자제)
  - 손절선 미설정 안내
  - 손절폭 10%+ 과도

#### 월간 리뷰 링크
대시보드 우상단 "월간 리뷰 →" 텍스트 링크 → `/reviews`

### 2.4 종목 상세 `/holdings/[ticker]`

- 헤더: 큰 종목명 + 시장 배지 (KOSPI/KOSDAQ) + 섹터 배지 + 티커
- **내 포지션 리본** (보유 중일 때만):
  - 평단 / 수량 / 손절 / 익절 + 미실현 수익률
- **시세 카드** (QuoteCard) — KIS 실시간
- **AI 분석 패널** (AnalyzePanel):
  - **다시 분석** 버튼 → Claude Haiku 호출
  - **카카오톡으로 보내기** 버튼 (분석 결과를 자기 카톡으로)
  - 결과: 권고 · 확신도 · 근거 3줄 · 실행계획 · 리스크
  - 캐시: 1시간 내 동일 시세면 재호출 없이 캐시

### 2.5 스크리너 `/screener`

#### 헤더
- 제목 + 설명 + 우측 **📊 성과** 버튼 → `/screener/performance`

#### 안내 배너
"참고용 리스트, 매수 권유 아님"

#### Run 메타 카드
- 최근 실행: "2시간 전" relative time (hover 시 absolute)
- 유니버스 N개 → 필터 M개 → 추림 3개
- 모델 + 비용

#### Pick 카드 3개
각각:
- 좌상단: #순위 + 종목명 + 티커 + **전략 배지** (🚀 돌파 / 🪂 저점)
- 우상단: 확신도 배지
- 상태 배지: 진입 도달 / Thesis 무너짐 / 기간 만료 / 포트 추가됨
- D-N (만료까지 남은 일수)
- **Thesis** 3줄
- **5일 수급** 줄: 외국인 +X만주 N/5일 + 기관 +Y만주 M/5일
- **추적** 줄 (outcome 추적 시작 후): 현재 +X% · 최고 +Y% · [익절 도달] 등
- **진입 참고 / 손절선 / 익절선** (3분할)
- **주의해야 할 리스크** 2~3개
- 하단 버튼:
  - **포트폴리오에 담기** (primary) — Sheet 자동 오픈, 값 채워짐
  - **☆ 관심** / **★ 관심 중** 토글
  - **상세 분석** (`/holdings/[ticker]`)

#### 관심 토글 효과
- ★ 표시 → 다음 30분 cron 부터 watch 리스트 진입
- 진입가 ±2% 도달 → "[진입 검토]" 카카오톡
- 손절선 통과 → "[Pick 무효]" 카카오톡 + status=invalidated
- 7일 경과 → "[Pick 만료]" + status=expired

### 2.6 스크리너 성과 `/screener/performance`
최근 90일 pick 통계:
- 4 Stat 카드: 총 pick / 확정됨 / 평균 수익률 / 승률
- **도달 분포 바**: 익절(빨강) / 손절(파랑) / 미도달(회색) 비율
- 각 pick row: 진입·손절·익절선 + 현재 수익률 + marker(익절/손절/진입/미도달)
- 클릭 시 `/holdings/[ticker]` 이동

### 2.7 월간 리뷰 `/reviews` · `/reviews/[id]`
- 리스트: 월별 카드 (생성월 / 보유종목수 / pick수 / 알림수 / 비용)
- 상세: 마크다운 렌더 (헤더/리스트/굵게 지원)
  - 이번 달 성과 / 보유 종목 건강도 / 스크리너 회고 / 다음 달 제안 / 리스크 체크리스트

### 2.8 홈 `/`
랜딩 페이지:
- Hero: 슬로건 + 포트폴리오/스크리너 진입 버튼
- 주요 기능 그리드 + 빠른 분석 5종목 + 세팅 현황

### 2.9 카카오 연결 `/login`
- 연결 안내 카드 (혜택 3가지: 손절·익절 알림 / 진입 타이밍 / Thesis 무너짐)
- 연결됨/미연결 상태별 다른 버튼
- 프라이버시 fine print

## 3. 자동 작동

### 3.1 카카오 알림 종류 (모두 자동)
| Type | 트리거 |
|---|---|
| `hit_stop` | 현재가 ≤ 손절선 |
| `near_stop` | 손절선 +3% 이내 |
| `hit_take` | 현재가 ≥ 익절선 |
| `near_take` | 익절선 -3% 이내 |
| `pick_entry_ready` | 관심 pick 의 진입가 ±2% 도달 |
| `pick_invalidated` | 관심 pick 의 손절선 통과 |
| `pick_expired` | 관심 pick 7일 경과 |
| `portfolio_mdd` | 포트 전체 피크 대비 -10% |
| `overweight` | 단일 종목 총 자산 25% 초과 |

### 3.2 자동 cron 일정
| 이름 | 주기 | 역할 |
|---|---|---|
| 가격 모니터링 | 평일 09:00~15:30 30분 | 손절/익절/MDD/비중 스캔 + 카카오 발송 |
| 스크리너 | 평일 매일 16:00 KST | 3개 pick 생성 |
| 월간 리뷰 | 매월 1일 09:00 KST | Opus 회고 |
| 종목 카탈로그 | 월 1회 | KRX 2,700개 동기화 |
| 자동 튜닝 | 주 1회 일요일 09:00 KST | 필터 임계값 분석 (드라이런/적용) |

### 3.3 카카오 메시지 예시
```
[손절 근접] 삼성에스디에스 (018260)

현재가 178,500원 (-1.2%)
손절선 175,000원
익절선 195,000원

손절선 근접 — 포지션 점검 후 대응 준비하세요.

※ 투자 참고용. 최종 판단은 직접.
```
→ **상세 보기** 버튼 클릭 시 `/holdings/018260`

## 4. 일상 사용 시나리오

### 시나리오 A: 새 종목 매수 → 자동 모니터링
1. 증권사 앱에서 매수 체결
2. `/dashboard` → ⊕ → 종목 추가 (손절·익절선 필수)
3. 이후 별 작업 X — cron 이 자동 감시
4. 손절·익절 근접 시 카카오톡 수신 → 직접 매매

### 시나리오 B: 추가 매수
1. 종목 카드 클릭 → 펼침 → **+ 추매**
2. 수량 + 가격 입력 → 새 평단 미리보기
3. 확정 → 토스트 알림

### 시나리오 C: 스크리너 종목 담기
1. 평일 16:30 경 `/screener` 접속
2. 마음에 드는 pick 의 **☆ 관심** 토글 → ★ 변경 → 알림 안내 토스트
3. 진입가 근처 도달 → 카카오톡 "[진입 검토]"
4. 매수 결정 시 **포트폴리오에 담기** → Sheet 자동 오픈 → 추가

### 시나리오 D: 월간 회고
1. 매월 1일 자동 생성된 리뷰 카카오 알림 (없음 — 직접 방문)
2. `/reviews` → 새 월 카드 클릭
3. 5섹션 정독: 성과 / 종목 건강도 / 스크리너 회고 / 다음 달 제안 / 리스크
4. 제안 받아들이면 대시보드에서 손절·익절 조정

### 시나리오 E: 빠른 종목 분석
1. `/` 또는 ⚙ → 종목찾기 탭
2. 빠른 분석 5종목 중 클릭 또는 검색
3. 상세 페이지 → **다시 분석**

## 5. AI 분석 결과 읽기

### 5.1 권고 (recommendation)
| 값 | 의미 |
|---|---|
| `hold` | 보유 유지 |
| `partial_buy` | 일부 추가매수 |
| `partial_sell` | 일부 매도 |
| `full_sell` | 전량 매도 |

> 원칙: 부분 매매 우선, 전량은 신중

### 5.2 확신도 (confidence)
- **high**: 신호 일치 (수급+차트+펀더)
- **medium**: 혼재된 신호
- **low**: 불확실, 보수적 접근

### 5.3 실행 계획 (action_plan)
- `immediate`: 오늘 할 행동
- `stop_loss`: 권장 손절선 (원)
- `take_profit`: 권장 익절선 (원)
- `review_at`: 다음 점검 시점

### 5.4 리스크 (risks)
주의 시나리오 2~3개. 발생 시 손절선 재검토.

### 5.5 스크리너 Thesis 읽기
- 전략 배지로 어떤 패턴인지 판단:
  - 🚀 **돌파** — 박스권 돌파 후 상승 기대 (박병창식)
  - 🪂 **저점** — 조정 끝 반등 시작
- 5일 수급 pill 로 외국인/기관 매수 강도 확인
- 추적 줄로 발표 이후 가격 흐름 즉시 파악

## 6. 트러블슈팅

### 카카오 알림 안 옴
1. `/login` 접속 → "현재 연결됨" 확인
2. 안 됐으면 **카카오 재연결** → 권한 다시 동의
3. 손절·익절선 설정 여부 확인 (`/dashboard` 카드 펼침)
4. 카카오톡 자체 수신 거부 설정 확인

### 비중 경고가 과도하게 뜸
- **현금 미입력**이 거의 항상 원인
- 💰 현금 pill 클릭 → 예수금 입력 → 저장
- 총 자산 기준으로 비중이 정확해짐

### "다시 분석" 응답 없음
- 캐시 (1시간 내 동일 시세) 면 즉시 응답
- 1~2초 정상, 5초+ 무응답이면 새로고침 후 재시도

### 스크리너 결과 없음
- "아직 실행 기록이 없습니다" — 초기 상태
- 평일 16:30 이후 또는 GitHub Actions 수동 실행

### 종목명이 티커로만 표시
- `stocks` 카탈로그 미적재 또는 동기화 안 됨
- 월 1회 자동, 또는 GitHub Actions → Load KRX Stocks 수동 실행

### 모바일에서 하단 탭바가 콘텐츠 가림
- 의도된 패딩 적용됨 (콘텐츠 `pb-20`)
- 안 보이면 새로고침

### 시세 0 또는 미표시
- KIS API 호출 실패 — Vercel env 의 `KIS_APP_KEY`/`KIS_APP_SECRET` 확인
- 장 시간 외엔 직전 종가 기준 표시 가능

## 7. 자주 묻는 질문

### Q1. 실제 매매가 자동으로 일어나나요?
**아니요.** 분석·알림 도구. KIS 주문 API 의도적 미구현. 매매는 본인이 증권사 앱에서.

### Q2. 카카오 알림이 친구한테 가나요?
**아니요.** "나에게 보내기" 권한만 사용.

### Q3. 비용은?
- Claude (Haiku 일 1회 + Opus 월 1회): **월 300~500원**
- KIS · FDR · GitHub Actions · Vercel Hobby · Supabase Free: 무료
- 총 월 ~500원 (예산 3,000원 대비 17%)

### Q4. 데이터 안전한가요?
- 환경변수 Vercel/Supabase 비밀 저장소
- RLS 적용, 단일 사용자
- 매매 권한 없음

### Q5. 스크리너 pick 따라 사면 수익이 나나요?
**보장 없음.** 참고용. 알고리즘 품질은 시간 누적 후 `/screener/performance` 로 검증.

### Q6. AI 분석이 틀렸을 때?
- 모든 응답은 참고. 시장은 예측 불가
- 손절선이 최후 안전망 — 무조건 지킬 것

### Q7. 자동 튜닝은 언제 작동?
- 인프라 완료, 데이터 누적 대기 중
- 30+ finalized pick 쌓이면 첫 dryrun
- 60~90+ 쌓이면 첫 자동 적용 가능

### Q8. 월간 리뷰 비용?
- Opus 4.7 × 월 1회 = **약 85원/월**

### Q9. 현금 비중 적정 범위?
- Bull 시장: 10~20%
- Bear 시장: 20~30%
- 10% 미만이면 시스템이 자동 경고

### Q10. 스크리너 매일 보내면 알림 폭주?
- 일 1회 cron 이라도 ★관심 표시한 pick 만 알림
- 신호 없는 날엔 자동 스킵 (no_signal)
- 하루 60건 카카오 한도 내

## 8. 시스템에 내장된 핵심 원칙

1. **잃지 않는 투자** — 원금 보호 > 수익 추구
2. **부분 매매** — 전량 지양
3. **구체 가격** — 손절·익절 반드시 명시
4. **확신도 표기** — high/medium/low 항상
5. **참고용 명시** — 매수 권유 아님
6. **시장 게이트** — 약세장엔 돌파 전략 자동 스킵
7. **분산 강제** — 단일 종목 25% 초과 경고

## 9. 데이터 표기 관례

- **상승 = 빨강, 하락 = 파랑** (한국 시장 관례)
- 가격: 원 단위 정수, 천 단위 쉼표
- 수익률: 소수 둘째 자리 + %
- 시가총액: "1.29조원" / "12,900억원"
- 시각: KST, 한국어 포맷
- 큰 금액: "1.2억원" / "5,000만원" (formatCompactKrw)

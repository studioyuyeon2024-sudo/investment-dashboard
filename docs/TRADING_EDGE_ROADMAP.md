## 0. 핵심 진단 (왜 현재 코드로는 부족한가)

리서치(학술/실무 출처) + 코드 실측으로 확인된 가장 시급한 사실:

> **현재 백테스트 평균 +0.77% 는 거래비용(왕복 ~0.5%) 차감 + 생존편향(연 1~4%p) 보정 시 사실상 0 또는 음수일 가능성이 높다.**

즉, 신호를 더 정교하게 만들기 전에 **"비용 차감 후 진짜 엣지가 있는가"** 부터 답해야 한다. 이것이 "잃지 않는 투자" 철학과 정확히 일치 — 엣지 없으면 베팅하지 않고, 있어도 작게 베팅하며, 통계가 확인될 때까지 의심한다.

### 시스템의 3대 빈 축
1. **체결 현실성** — 거래비용·슬리피지·유동성 무시 (백테스트 ≠ 실전)
2. **통계적 신뢰** — 소표본 승률을 과신, 과적합 위험
3. **자금 관리** — "얼마를 살지" 가 전무 (실전 수익률 분산 최대 결정 요인)

## 1. 완료된 개선 (이번 세션, PR #44~#47)

### ✅ PR #44 — 백테스트 실전화
- 진입가: 신호 다음날 **시가** (look-ahead 최소화)
- **ATR(14) 손절/익절 day-by-day 시뮬레이션** (기존: 무조건 21일 보유)
  - 손절 = 진입 − 2×ATR, 익절 = 진입 + 3×ATR
  - 갭하락 시가 청산, 동시 터치 시 보수적 손절 우선
- **거래비용 왕복 0.50%** 차감 (거래세 0.20% + 수수료 + 슬리피지)
- **유동성 필터** 20일 평균 거래대금 ≥ 10억원
- 통계 지표: 승률 95% 신뢰구간, Profit Factor, Sharpe, 청산사유 분포

### ✅ PR #45 — ATR 손절/익절 룰 일관화
- 스크리너가 ATR 기반 룰 값(2×/3×ATR)을 기본 제시
- Claude 는 구조적 근거 있을 때만 조정, 누락 시 폴백
- 백테스트와 실전 스크리너가 **동일 손절/익절 산식** → 백테스트 예측력 확보

### ✅ PR #46 — 유동성 게이트
- 실전 스크리너 `quant_filter` 에 거래대금 10억 하한
- 백테스트와 일치 (체결 안 되는 종목 사전 제외)

### ✅ PR #47 — 자가학습 소표본 가드

### ✅ 성과 페이지 지표 보강

| 시점 | 증권거래세 |
|---|---|
| 과거 | 0.18% |
| 2025 | 0.15% |

백테스트 비용 모델은 2026 기준 0.20% + 수수료 + 슬리피지 = 왕복 0.50% 적용. 코스닥 소형주는 슬리피지가 커서 실제 0.5~0.7%.

## 3. 남은 개선 (우선순위 순)

### 🥇 1순위 · 포지션 사이징 (미구현, 영향도 ★★★★★)
"얼마를 살지" 가 실전 수익률 분산의 최대 결정 요인인데 비어 있음.

**구현 방안**:
- 종목당 리스크 1% 룰: `수량 = (자본 × 1%) ÷ (진입가 − 손절가)`
- ATR 손절과 결합 → 변동성 큰 종목 자동으로 작게 매수
- 포트 히트 캡 6% (동시 최대 6종목 × 1%)
- bear 시장이면 캡 3% 로 축소
- 동일 테마/업종 합산 리스크 1.5~2배 계산 (상관관계 가드)

**적용 위치**: 대시보드 종목 추가 폼 + pick 카드에 "권장 수량" 표시. 계좌 자본 입력 필드 필요(현금 필드 활용 가능).

### 🥈 2순위 · 진입·청산 정교화 (부분 구현, ★★★★)
- ✅ ATR 손절 (완료)
- ⬜ **부분 익절**: 1.5R 에서 50% 청산 + 잔량 손절을 본전으로 이동
- ⬜ **트레일링 스탑**: 잔량은 종가 − 2.5×ATR 또는 MA20 이탈
- ⬜ 진입: buy-stop-limit (+2% 상한), 갭상승 +2% 초과 시 스킵

### 🥉 3순위 · 신호 데이터 보강 (미구현, ★★★★)
- ⬜ **20일 누적 외국인·기관 순매수** (현재 5일만) — 추세 확인
- ⬜ **RS rating** (자체 산출): 유니버스 내 12개월 수익률 백분위, RS<70 제외
- ⬜ 개인 단독 순매수 우위 종목 감점 (개인 −0.7 역상관)

### 4순위 · 시장환경·이벤트 (미구현, ★★★)
- ⬜ 시장폭 (advance/decline) + VKOSPI 게이트
- ⬜ DART 어닝 캘린더 회피 (실적발표 ±5거래일)
- ⬜ 공매도 잔고 급증 종목 감점
- ⬜ 코스닥 변동성 차등 (더 큰 ATR 배수, 더 작은 사이즈)

### 5순위 · 통계 검증 고도화 (부분 구현, ★★★)
- ✅ 승률 신뢰구간, Profit Factor (완료)
- ⬜ Walk-forward analysis (in-sample/out-of-sample 롤링)
- ⬜ Deflated Sharpe Ratio (다중시도 보정)
- ⬜ 파라미터 둔감성(plateau) 테스트
- ⬜ 합격 게이트: OOS Profit Factor ≥ 1.5 & 비용후 expectancy > 0

## 4. "백테스트 착시" 7대 함정 (항상 경계)

1. **비용 누락**: +0.77% 는 비용 차감 시 ~0%. 가장 흔하고 치명적 → ✅ 해결
2. **생존 편향**: 현재 지수 구성으로 과거 봄 → 연 1~4%p 과대평가 → ⬜ point-in-time 유니버스 필요
3. **룩어헤드**: 당일 종가 신호 → 당일 체결 가정 → ✅ 다음날 시가로 해결
4. **완전체결 가정**: 소형주 무한 체결 가정 → ✅ 유동성 필터로 완화
5. **다중시도 과적합**: 여러 파라미터 중 최고를 고름 → ⬜ Deflated Sharpe 필요
6. **소표본 과신**: 50거래 승률 ±14%p → ✅ 신뢰구간 라벨로 해결
7. **상관관계 무시**: 같은 테마 6종목 = 1개 큰 베팅 → ⬜ 포지션 사이징의 테마 가드

## 5. 실행 순서 (다음 세션용)

```
현재 위치: 1단계(비용·편향 보정) 대부분 완료
다음: 2단계 자금관리 → 3단계 통계 게이트 → 4단계 신호 고도화 → 5단계 시장환경
```

### 다음 세션 즉시 착수 권장
1. **백테스트 재실행** — 비용·손절 반영 후 expectancy 가 양수인지 확인 (가장 중요)
   - 음수면: 전략 자체 재설계 (현 필터로는 실전 불가 판정)
   - 양수면: 포지션 사이징(2순위)으로 진행
2. **포지션 사이징** 구현 (1% 리스크 룰)
3. **20일 수급 + RS rating** 추가

### 검증 게이트 (실전 투입 조건)
전략이 실거래로 갈 자격:
- [ ] 비용 차감 후 expectancy > 0
- [ ] Profit Factor ≥ 1.5
- [ ] 표본 ≥ 30 (이상적으로 100)
- [ ] Walk-forward OOS 에서 IS 의 50% 이상 유지

하나라도 미달이면 **신호 생성은 하되 실매매 보류**.
